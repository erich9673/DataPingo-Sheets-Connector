Overview

DataPingo Sheets Connector uses certain third-party sub-processors to provide our service. This page lists all sub-processors that may process customer data as part of our service delivery.

Data Minimization: We only share the minimum data necessary with sub-processors to provide our service functionality. No sub-processor receives complete customer spreadsheet data.

Current Sub-Processors

Sub-Processor Purpose Data Processed Location Security Standards
Google LLC Google Sheets API access for spreadsheet monitoring Spreadsheet cell values (as specified by user), OAuth tokens United States SOC 2, ISO 27001, GDPR compliant
Railway Corp Cloud hosting and infrastructure services Application logs, temporary cached data United States SOC 2, encrypted infrastructure
Slack Technologies Notification delivery via incoming webhooks Formatted notification messages only United States SOC 2, ISO 27001, GDPR compliant

Data Processing Safeguards

  • Minimal Data Sharing: Only essential data required for service functionality is shared
  • Encrypted Transmission: All data is transmitted using HTTPS/TLS encryption
  • Temporary Processing: Data is processed temporarily and not permanently stored by sub-processors
  • User Control: Users control what data is monitored and where notifications are sent
  • Secure Authentication: OAuth 2.0 standards used for all API access

Sub-Processor Requirements

All sub-processors must meet the following requirements:

  • Maintain appropriate technical and organizational security measures
  • Process data only for the specific purposes outlined in our agreement
  • Comply with applicable data protection laws (GDPR, CCPA, etc.)
  • Implement data encryption in transit and at rest where applicable
  • Provide audit capabilities and compliance documentation
  • Notify us of any security incidents or data breaches

Changes to Sub-Processors

We will:

  • Update this page when adding, removing, or changing sub-processors
  • Ensure any new sub-processors meet our security and privacy requirements
  • Maintain the same level of data protection with any new sub-processors
  • Notify customers of material changes to sub-processor arrangements

Customer Rights

Customers have the right to:

  • Request information about our sub-processors and their data processing activities
  • Object to the use of specific sub-processors (may affect service availability)
  • Request deletion of their data from all sub-processors
  • Receive notification of changes to our sub-processor arrangements
